# AI-Powered Pattern Insights Feature

## Overview
The AI-Powered Pattern Insights feature provides educational research insights about congressional trading patterns using machine learning analysis. This feature is designed for transparency, accountability research, and investigative journalism.

## Features

### 1. Timing Pattern Detection
AI analyzes correlations between congressional trades and legislative events:
- Detects trades occurring before committee hearings, votes, and legislative milestones
- Calculates confidence scores based on timing proximity and trade size
- Identifies potential information advantages

**Example Insight:**
- Nancy Pelosi NVDA purchase $3M, 65 days before AI Safety Act committee vote
- AI Confidence: 92%
- Pattern: Large tech purchase before major AI legislation

### 2. Committee-Trading Correlations
Machine learning analysis of oversight alignment:
- Tech Oversight ‚Üí Tech Stock Trading (87% correlation)
- Banking Committee ‚Üí Financial Stock Trading (82% correlation)
- Energy Committee ‚Üí Energy Stock Trading (91% correlation)

**Research Finding:** Committee members with sector oversight show high concentration in stocks they regulate.

### 3. Anomaly Detection
Statistical outlier identification:
- Exceptionally large trade volumes (>2 standard deviations)
- STOCK Act filing violations (>45 days)
- Unusual filing delay patterns
- Z-score calculations for statistical significance

### 4. Sector Concentration Analysis
ML-powered sector trend detection:
- Technology Sector: $5.2M (67% of volume)
- Financial Sector: $437K
- Energy Sector: $407K
- Sector diversity index: 0.42 (low diversity suggests information clustering)

### 5. Research Questions Generated by AI
Machine learning identifies areas for further investigation:
1. Do committee members have systematic information advantages?
2. Do committee leaders show different trading patterns?
3. Are there partisan differences in trading patterns?

### 6. Transparency & Accountability Metrics
AI-calculated transparency scoring:
- Overall Transparency Grade: B
- Compliance Rate: 91%
- Data Quality Score: 76%
- Average Filing Delay: 38 days

## Technical Implementation

### Backend: Pattern Insights Engine
**File:** `src/analysis/pattern_insights_engine.py`

**Key Classes:**
- `PatternInsightsEngine`: Main analysis engine
  - `generate_comprehensive_insights()`: Full analysis pipeline
  - `_analyze_timing_patterns()`: Legislative event correlation
  - `_analyze_committee_correlations()`: Oversight-trading alignment
  - `_detect_unusual_activity()`: Statistical anomaly detection
  - `_analyze_sector_trends()`: Sector concentration analysis
  - `_calculate_transparency_metrics()`: Transparency scoring

**Dependencies:**
- pandas: Data manipulation
- numpy: Statistical calculations
- json: Export insights

**Output Format:**
```json
{

}
```
